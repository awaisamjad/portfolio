## Motivation
I wanted to understand more about how the terminal works, so I decided to write my own shell for personal use. I hope to make it good enough to be a personal substitute for Bash and Nushell.

### Technologies Used
**Language** - **C**

**Build Tool** - **Make**


### Features and Functionality



**The usual commands**

- **cd** - Change Directory
- **mkd** - Make Directory
- **touch** - Create file

### Planned Features and Functionality
- **Piping** - Allows you to combine commands
- **Scripting** - Allow you to combine commands and customise the shell to your liking
